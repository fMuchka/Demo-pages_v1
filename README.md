# demo-pages-v1

Demo of: 
 - Vue (2.0)
 - Vuetify
 - API fetch data
 - animations (GSAP)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run prod
```

### Lints and fixes files

```
npm run lint
```
